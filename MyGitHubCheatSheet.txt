1. to change origin (repository) url

    if remote repository not added as a master 
        git remote add origin <https://github.com/user/repository.git>
    else
        git remote set-url origin <https://github.com/user/repository.git>
        git remote add origin <https://github.com/user/repository.git>

3. to add all files [recursively] within repository folders
    git add .
    
2. to commit all changes to mirror local directory
    git commit -a -m '<message>'

3. to push changes git push -u origin <branch>

4. To temp store password/username per repo
    git config credential.helper store
    git push https://github.com/repo.git

    Username for 'https://github.com': <USERNAME>
    Password for 'https://USERNAME@github.com': <PASSWORD>
    
   to set when cache should expire in seconds
    git config --global credential.helper 'cache --timeout <seconds>'
    
5. To checkout and make a new branch
    git checkout -b '<name>'
    
6. To switch to a different branch
    git checkout '<name>'
